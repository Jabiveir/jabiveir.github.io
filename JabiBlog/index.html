JABIVEIR 3RD ANNIVERSARY!!
First of all: sorry for not posting for so long,,, Rest assured, however, that I was working VERY hard in that time!
Secondly: happy Jabiveir Day!!! Today is June 17th, the anniversary of the release of Jabiveir Quest and this website as a whole. Since it came out in 2022, that marks three years of Jabiveir! And, as I hope will become tradition, I have some pretty big things prepared. The first of which is,,,

THE RELEASE OF JABICHAT!!
That's right, after around half a year in development, JabiChat is FINALLY ready for public release.
Of course, this doesn't mean that it's done. There are still a ton of things that I want to add (and bugs to fix immediately after release, I imagine), but it now is ready enough to be publicly used, and thus I'm probably gonna take a break from it for a while (after fixing all the release-day bugs).
You can start using JabiChat NOW from the main site page or by putting https://chat.jabiveir.com into your URL bar!

RSS Support
If JabiChat won't be enough to notify you of website news, then you can use the newly added Jabiveir RSS feed!
If you don't have an RSS aggregator yet, then you should get one immediately, because they are awesome. I recommend QuiteRSS (download here) because it's free, simple, and has a lovely aesthetic!
To use this site's RSS feed, simply copy the RSS link (found on the main site page) into your RSS aggregator (that's how it works for QuiteRSS at least, I'm uncertain about other programs).

Future Plans
As you may have seen on the main site, the Jabiveir Wiki has been blocked off for renovations. That is because, with JabiChat finally being ready for release, I'm finally free to move on to updating the wiki into Jabipedia! My plans for the pages have changed, so I will be rewriting most of it, and I want to finish all of it before release, so it will unfortunately be a little while before it's done (though not seven months! Hopefully,,,); I've already been making fairly steady progress, though. This release will also be accompanied by minor updates to each of my previous games, for reasons that will become obvious when the update arrives. Past that, I'd like to update the main site page to look better (with a different format for mobile users), which shouldn't be too difficult.

JabiWiki and Other Plans
The Wiki page for "Five Nights at Jabiveir's" is now out!

That certainly isn't enough for a blog post on its own, so I'd also like to discuss my specific plans for the Wiki, as well as some changes to my plans for JabiChat. For the Wiki, I have several pages that I still want to make, including one for the rest of my games, a page for upcoming and cancelled projects, a page just about Jabiveir in general, and a page for the Jabiveir Website itself. The main reason that I want to change my plans a bit is mainly because of the Jabiveir Website page, as I intend for it to include a list of site-wide achievements. I have a few ideas for what achievements I would want to put in, but implementing the achievements is likely going to be the most time-consuming part of the entire Jabiveir Wiki's development.

This brings me to my new plan: my main focus is still going to be on building the Wiki, but before I continue with that, I will try to put up a rudimentary alpha version of JabiChat, and the two will then be developed in parallel. This is certainly going to make it take longer for both to be developed than if I were to just work on one and then the other, but I think it's important to get JabiChat up and running in some capacity as soon as is feasible, because most of the bugs are going to be ones that I'm only going to be able to find by having people use the application, so I would want people using it as soon as I can manage; also, this gives more opportunity for user suggestions to be baked into the foundation of the application, which will make it much better in the end.

I have no clue how long it will take to build JabiChat's alpha, but I'll be working on it for (hopefully) most of this weekend, and I'll be sure to make another post when it's out.
